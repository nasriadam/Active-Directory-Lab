# Active Directory Lab

## Objective

The Active Directory Lab project aimed to build and configure a controlled Windows enterprise environment to understand the fundamentals of **Microsoft Active Directory Domain Services (AD DS)**.

The objective was to deploy a domain infrastructure, configure centralized user management, manage domain computers, and practice basic Windows Server administration in a simulated corporate environment.

The lab environment includes:
- Windows Server 2022 as a Domain Controller
- Active Directory Domain Services (AD DS)
- DNS Server
- Windows 10 domain-joined clients

---

## Skills Learned

- Installation and configuration of Active Directory Domain Services.
- Deployment of a Windows Server Domain Controller.
- Configuration of DNS services for Active Directory.
- User, group, and computer account management.
- Organizational Units (OUs) administration.
- Joining Windows clients to an Active Directory domain.
- Understanding centralized authentication and domain management.
- Basic Group Policy administration.

---

## Tools Used

- **Windows Server 2022**
  - Active Directory Domain Services
  - DNS Server
  - Group Policy Management

- **Windows 10**
  - Domain client machines

- **VirtualBox**
  - Virtualization environment
